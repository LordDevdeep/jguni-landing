# Submission Guide

## Demo Video Script

Use this as a 2-5 minute walkthrough script.

### 1. Introduction

Hello, my name is [Your Name]. This is my internship assignment submission: a modern landing page redesign inspired by JG University.

The project is built using Next.js and Tailwind CSS, with Framer Motion for animations and interactions.

### 2. Desktop View

Start at the top of the page.

This is the hero section. I redesigned the page with a clean university-style visual identity instead of a generic template look. The navbar is fixed, responsive, and includes smooth section navigation.

The page uses the JG University logo and a warm institutional color palette with campus imagery.

### 3. About Section

Scroll to the About section.

This section explains the university positioning, inspired by the original JG University website: experiential learning, knowledge beyond books, academic foundation, and student-first support.

### 4. Programs and Interactions

Scroll to the programs section.

The programs section is interactive. Users can select a program and instantly see the school, duration, credential, career direction, and focus areas. This improves the user experience compared to a static card grid.

### 5. Features, Campus, Testimonials

Scroll through the strengths, campus, stats, and testimonials sections.

The strengths section highlights the academic ecosystem. The campus section uses visual cards and real imagery. The testimonials section includes a carousel with navigation controls.

### 6. CTA and Footer

Scroll to the CTA section and footer.

The CTA guides students toward applying or downloading the brochure. The footer includes key links and contact information.

### 7. Mobile Responsive View

Open browser dev tools or resize the screen to mobile width.

Show that the navbar becomes a mobile menu, the hero stacks properly, program cards remain usable, company logos fit, and all sections are readable on smaller screens.

### 8. Closing

This completes the walkthrough. The project is responsive, visually modern, and built with clean Next.js and Tailwind CSS structure.
